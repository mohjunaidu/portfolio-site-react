# Muhammad Junaidu Ismail — Portfolio (React)

Sleek modern portfolio built with React, Vite, Tailwind & Framer Motion.

## Run locally
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Deploy
- Deploy on Vercel (recommended) or Netlify.
- Build: `npm run build`
- Output directory: `dist`

## Notes
- CV: Place your CV file as `public/cv.pdf` (it will be served at `/cv.pdf`)
- Contact uses EmailJS — replace SERVICE_ID/TEMPLATE_ID/USER_ID in `Contact.tsx`
